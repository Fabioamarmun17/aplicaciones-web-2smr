¿Qué es el HTML5?

HTML5 es la versión más reciente del lenguaje de marcado utilizado para crear páginas web. Es más semántico, accesible y soporta contenido multimedia sin necesidad de plugins externos como Flash...

¿Qué es CSS3?

CSS3 es el lenguaje utilizado para estilar las páginas web. Permite definir colores, tamaños, márgenes yy otros aspectos visuales que hacen que las páginas web luzcan mejor.
